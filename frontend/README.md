# JS Learning Platform 🎓

Интерактивная платформа для обучения JavaScript. Проект разработан в рамках дипломной работы.

## 🚀 О проекте
Данная платформа предназначена для закрепления знаний по JavaScript. Она включает в себя теоретические уроки, систему тестирования (квизы) и личный кабинет пользователя для отслеживания прогресса.

### Основные функции:
* **Авторизация и профиль:** Регистрация пользователей и сохранение прогресса.
* **Интерактивные уроки:** Структурированный контент по JS.
* **Система квизов:** Проверка знаний после прохождения модулей.
* **Дашборд:** Визуализация прогресса обучения.
* **Геймификация:** Модальные окна достижений и система уровней.

## 🛠 Технологии
Проект построен на современном стеке:
* **Frontend:** [React.js](https://reactjs.org/)
* **Стилизация:** [Tailwind CSS](https://tailwindcss.com/)
* **Управление состоянием:** [Zustand](https://github.com/pmndrs/zustand)
* **Маршрутизация:** React Router
